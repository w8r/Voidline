✔ (August 2013 - October 2013) Brainstorm, set the project goals, and research various ways to gather and display the knowledge base

✔ (November 2013) Build a basic website, providing basic information about the project, its goals, and get feedback from the community

- illustrate the project's goals and expected behavior in a more direct way, via illustrations and videos

- submit the project to a mass crowdfunding service such as Kickstarter, or IndieGoGo

- promote the crowdfunding campaign until it reaches the required funding, and carefully gather community feedback

- start development of the project

- create an advanced taxonomy system that is able to accurately classify every thing, and describe its properties and behaviors

- import/parse existing knowledge base(s) such as DBpedia or Freebase, to provide an initial knowledge database that can be improved upon

- create an intuitive, interactive pan-and-zoom map to visualize the data, that would be efficient and easy to use

- implement proper search and research tools applicable on the data, that would use natural language

- implement knowledge engine that would be able to compute and deduce facts from data

- implement a social platform that would allow users to contribute to the knowledge base, interact with it and with each other

- perform a closed sandbox beta test of the system, to find and fix flaws, and improve the system

- launch to the public
